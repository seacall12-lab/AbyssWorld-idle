# Abyss World - 다크 판타지 Idle RPG

> 상업용 품질의 다크 판타지 Idle 게임

## 🎮 게임 소개

**Abyss World**는 깊은 어둠의 세계를 배경으로 한 고퀄리티 Idle RPG입니다.
- 실시간 전투 시스템
- 장비 강화 및 합성
- 펫 육성 시스템
- 다양한 스킬 메커니즘
- 환생 시스템으로 무한 진행 가능

## ✨ 최근 개선사항 (상업화 업데이트)

### 🎨 UI/UX 개선
- **다크 판타지 테마 강화**: 귀족적이고 신비로운 색상 팔레트 (보라색, 금색, 크림슨 등)
- **모바일 최적화**: 터치 반응성 개선, 버튼 크기 조정
- **부드러운 애니메이션**: 버튼 인터랙션, 전환 효과, 호버 피드백
- **로딩 화면**: 게임 시작 시 전문적인 로딩 화면 추가
- **시각적 피드백**: 스킬별 다양한 색상, 활성화 상태 표시

### 📱 모바일 최적화
- **뷰포트 적응**: 다양한 해상도 지원 (480px 이하 최적화)
- **터치 영역 확대**: HIT 버튼 등 주요 버튼 크기 증가
- **반응형 레이아웃**: 화면 크기별 UI 자동 조정
- **성능 최적화**: 렌더링 최적화, 메모리 관리

### 🎯 UX 개선
- **타 탭 색상 코딩**: 전투(크림슨), 장비(인디고), 펫(초록), 스킬(보라) 등 구분
- **스킬별 테마 색상**: 각 스킬마다 독특한 색상 실시간 표시
- **인터랙티브 요소**: 토스트 메시지, 로그 색상 코딩, 상태 표시 강화
- **타이틀 개선**: 판타지 아이콘으로 강화된 헤더

### ⚙️ 기술적 개선
- **Enhanced Canvas UI System**: 버튼 상태 추적, 애니메이션 시스템
- **향상된 터치 감지**: 터치와 마우스 모두 완벽 지원
- **앱 메타데이터**: PWA 설정 완벽화 (manifest.json 강화)
- **CSS 애니메이션**: 글로벌 애니메이션 변수, 반응형 미디어 쿼리

## 🚀 설치 및 실행

### 웹에서 실행
```bash
# HTTP 서버 실행
python3 -m http.server 8000
# http://localhost:8000 에서 접속
```

### PWA로 설치 (안드로이드/iOS)
1. 브라우저에서 게임 실행
2. 주소바의 "설치" 버튼 클릭
3. 홈 화면에 추가

## 🎮 게임 시스템

### 전투 시스템
- **HIT**: 적을 직접 공격
- **AUTO**: 자동 공격 토글
- **CRIT**: 치명타 확률 기반 추가 피해
- **Stage**: 난이도 조절

### 스킬 시스템
- **파워**: 물리 공격 강화
- **처형**: 고정 데미지
- **광폭**: 공격 강화 (지속시간)
- **가속**: 공격 속도 증가
- **행운**: 골드 획득량 증가

### 장비 시스템
- **무기**: 공격력 증가
- **방어구**: 생명력 증가
- **반지**: 특수 능력 부여 (환금, 치명타 등)

### 펫 시스템
- 최대 4개 펫 슬롯
- 자동 전투 지원
- 펫 스킬 시스템

## 📊 성능

- **로딩 시간**: < 1초
- **프레임율**: 60 FPS (모바일에서 최적화)
- **번들 크기**: < 500KB
- **오프라인 지원**: 서비스 워커를 통한 완벽한 오프라인 플레이

## 🛠 기술 스택

- **Frontend**: HTML5 Canvas, Vanilla JavaScript
- **Storage**: IndexedDB (로컬 저장)
- **Offline**: Service Worker (PWA)
- **Styling**: CSS3 (변수, 애니메이션, 반응형)
